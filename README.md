# Rust examples

## Rust

Build [Rust](https://www.rust-lang.org/en-US/) project with [Cargo](http://doc.crates.io/guide.html):

    cargo build

Source files are in the `src` folder.

Executable files are in `target/debug` folder and can be run with

    ./path/to/executable
